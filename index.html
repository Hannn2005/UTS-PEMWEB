Pengenalan

HTML-CSS-Javascript-Tailwind

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. Dengan HTML, pengembang dapat menentukan struktur dasar dari sebuah situs, seperti menampilkan teks, gambar, tautan, tabel, hingga formulir. HTML bekerja menggunakan elemen atau tag yang memberi tahu browser bagaimana setiap bagian halaman harus ditampilkan. Bahasa ini menjadi fondasi utama dalam pengembangan web sebelum diberikan tampilan dengan CSS dan interaktivitas menggunakan JavaScript. Tanpa HTML, sebuah halaman web tidak akan memiliki kerangka atau struktur yang dapat dibaca oleh browser maupun pengguna.

CSS atau Cascading Style Sheets adalah bahasa yang digunakan untuk mengatur tampilan dan gaya dari halaman web yang dibuat dengan HTML. Dengan CSS, pengembang dapat mengontrol warna, ukuran teks, jarak antar elemen, tata letak, serta efek visual seperti animasi dan transisi. CSS memisahkan konten dari tampilan, sehingga kode menjadi lebih rapi, mudah dikelola, dan fleksibel dalam mengubah desain tanpa memodifikasi struktur HTML.

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas pada halaman web. Dengan JavaScript, sebuah website dapat merespons tindakan pengguna seperti klik, input data, atau pergerakan mouse secara dinamis. Selain itu, JavaScript juga dapat digunakan untuk memanipulasi elemen HTML dan CSS secara langsung, membuat animasi, hingga membangun aplikasi web yang kompleks dengan bantuan framework seperti React, Vue, atau Angular.

Tailwind CSS adalah framework CSS berbasis utility-first yang memungkinkan pengembang mendesain tampilan langsung di dalam markup HTML menggunakan kelas-kelas siap pakai. Dengan pendekatan ini, pengembang tidak perlu menulis CSS dari nol, melainkan cukup menggabungkan kelas seperti flex, text-center, atau bg-blue-500 untuk membuat desain dengan cepat dan konsisten. Tailwind mempermudah proses styling sekaligus menjaga struktur kode tetap efisien dan mudah diatur.

Membuat Website Sederhana

Website sederhana dapat dibuat menggunakan tiga komponen utama, yaitu HTML, CSS, dan JavaScript. HTML digunakan untuk membuat struktur halaman seperti header, isi, dan footer. CSS berfungsi untuk mempercantik tampilan agar lebih menarik dan rapi. Sedangkan JavaScript digunakan untuk menambahkan interaksi sederhana agar website terasa lebih hidup.

Langkah Pertama

Membuat HTML (HyperText Markup Language) adalah struktur utama dari sebuah website. File ini berfungsi untuk menampilkan teks, gambar, video, dan bagian-bagian halaman lainnya. Dengan cara membuat file dengan ekstensi .html

Langkah Kedua

Menambahkan CSS (Cascading Style Sheets) digunakan untuk menghias dan mengatur layout agar website terlihat rapi dan indah. Tanpa CSS, tampilan web hanya berupa teks polos. Dengan CSS, kamu bisa mengatur warna, ukuran, spasi, posisi, dan gaya huruf. Dengan cara membuat file dengan ekstensi .css

Langkah Ketiga

Menambahkan JavaScript (JS) membuat website lebih hidup dan interaktif. Kamu bisa menambahkan efek, animasi, atau logika tertentu pada website. Dengan cara membuat file dengan ekstensi .js
